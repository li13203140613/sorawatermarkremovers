## 八、性能优化

### 1. 图片优化
```
- 使用 Next.js Image 组件
- 懒加载: loading="lazy"
- 现代格式: WebP/AVIF
- 响应式: srcset
- 占位符: blur placeholder
```

### 2. 代码分割
```
- 动态导入: 视频播放器组件
- 路由级别: 每个页面独立打包
- 按需加载: 分类筛选逻辑
```

### 3. 缓存策略
```
- 静态资源: Cache-Control: public, max-age=31536000
- API 数据: ISR (Incremental Static Regeneration)
- 重新验证: revalidate: 3600 (1小时)
```
