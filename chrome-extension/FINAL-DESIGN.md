## 三、按钮状态变化

### Sora 页面下载按钮状态

#### 1. 默认状态（Default）
```
┏━━━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃  📥  Download     ┃  ← 渐变紫色
┃                   ┃     正常阴影
┗━━━━━━━━━━━━━━━━━━━┛
```

#### 2. 悬停状态（Hover）
```
┏━━━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃  📥  Download     ┃  ← 向上浮起 2px
┃                   ┃     阴影加强
┗━━━━━━━━━━━━━━━━━━━┛     轻微放大
```

#### 3. 点击中（Active）
```
┏━━━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃  📥  Download     ┃  ← 轻微缩小
┃                   ┃     scale(0.98)
┗━━━━━━━━━━━━━━━━━━━┛
```

#### 4. 下载中（Loading）
```
┏━━━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃  ⏳ Downloading...┃  ← 转圈动画
┃                   ┃     禁用点击
┗━━━━━━━━━━━━━━━━━━━┛
```

#### 5. 成功（Success）- 2秒后消失
```
┏━━━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃  ✅ Downloaded!   ┃  ← 绿色背景
┃                   ┃     淡出动画
┗━━━━━━━━━━━━━━━━━━━┛
```

#### 6. 失败（Error）
```
┏━━━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃  ❌ Failed        ┃  ← 红色背景
┃     Click retry   ┃     可重试
┗━━━━━━━━━━━━━━━━━━━┛
```

#### 7. 积分不足（No Credits）
```
┏━━━━━━━━━━━━━━━━━━━┓
┃                   ┃
┃  💎 No Credits    ┃  ← 灰色背景
┃   Click recharge  ┃     跳转充值
┗━━━━━━━━━━━━━━━━━━━┛
```

## 四、颜色和样式规范

### 颜色定义
```css
/* 主色 - 渐变紫 */
--primary-gradient: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 状态色 */
--success: #10b981;
--error: #ef4444;
--disabled: #9ca3af;

/* 中性色 */
--gray-bg: #f3f4f6;
--gray-text: #6b7280;
--gray-dark: #1f2937;

/* 提示色 */
--info-bg: #eff6ff;
--info-border: #3b82f6;
--info-text: #1e40af;
```

```css
/* 按钮阴影 */
--shadow-sm: 0 2px 8px rgba(0, 0, 0, 0.05);
--shadow-md: 0 4px 12px rgba(102, 126, 234, 0.3);
--shadow-lg: 0 6px 20px rgba(102, 126, 234, 0.45);

/* 卡片阴影 */
--card-shadow: 0 2px 8px rgba(0, 0, 0, 0.05);
```
