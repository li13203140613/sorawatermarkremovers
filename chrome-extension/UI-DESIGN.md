## 三、按钮状态设计

### 下载按钮的不同状态

#### 1. 默认状态
```
┏━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  📥  Download            ┃
┃      💎 1 Credit         ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

#### 2. Hover 状态（鼠标悬停）
```
┏━━━━━━━━━━━━━━━━━━━━━━━━━┓  ← 轻微放大
┃  📥  Download            ┃     阴影加强
┃      💎 1 Credit         ┃     向上浮起 2px
┗━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

#### 3. Loading 状态（下载中）
```
┏━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ⏳  Downloading...      ┃  ← 转圈动画
┃      Please wait         ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

#### 4. Success 状态（成功）
```
┏━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ✅  Downloaded!         ┃  ← 绿色背景
┃      1 Credit used       ┃     2秒后消失
┗━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

#### 5. Error 状态（失败）
```
┏━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  ❌  Failed              ┃  ← 红色背景
┃      Click to retry      ┃     显示错误
┗━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

#### 6. 积分不足状态
```
┏━━━━━━━━━━━━━━━━━━━━━━━━━┓
┃  💎  No Credits          ┃  ← 灰色/禁用
┃      Click to recharge   ┃     点击跳转充值
┗━━━━━━━━━━━━━━━━━━━━━━━━━┛
```

## 四、图标和颜色规范

### 图标选择
- **下载**: 📥 (推荐) / ⬇️ / 💾
- **积分**: 💎 (钻石)
- **充值**: 💳 (信用卡)
- **登录**: 🔐 (锁)
- **退出**: 🚪 (门)
- **刷新**: 🔄 (循环箭头)
- **成功**: ✅
- **失败**: ❌
- **加载**: ⏳
- **提示**: 💡

### 颜色方案
```css
/* 主色 - 渐变紫 */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* 金色 - 积分卡片 */
background: linear-gradient(135deg, #fef3c7 0%, #fde68a 100%);
border: 2px solid #fbbf24;

/* 成功 - 绿色 */
background: #10b981;

/* 失败 - 红色 */
background: #ef4444;

/* 禁用 - 灰色 */
background: #9ca3af;
```

## 五、动画效果

### 按钮点击动画
```
1. Normal:     scale(1)
2. Hover:      scale(1.02) + translateY(-2px)
3. Active:     scale(0.98)
4. Success:    弹跳动画 → 淡出
```

### 积分数字变化动画
```
从 3 → 2: 数字滚动向下，0.5秒
```

### 下载进度动画
```
Loading: 转圈图标旋转 360deg，1秒循环
```
