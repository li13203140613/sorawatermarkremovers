## Installation Steps

### Method 1: Install from Chrome Web Store (Recommended)

1. Visit [Chrome Web Store](#)
2. Search for "RemoveWM"
3. Click "Add to Chrome"
4. Click "Add Extension" in the popup

### Method 2: Manual Installation of Developer Version

If you want to use the latest test version:

#### Step 1: Download Extension Files

```bash
# Download extension source code
git clone https://github.com/removewm/chrome-extension.git
cd chrome-extension
npm install
npm run build
```

#### Step 2: Load into Chrome

1. Open Chrome browser
2. Visit `chrome://extensions/`
3. Enable "Developer mode" in the top right
4. Click "Load unpacked"
5. Select the `build` folder

## Usage Tutorial

### Basic Usage

#### 1. Login

First-time use requires login:

1. Click the RemoveWM icon in browser toolbar
2. Click "Login" button
3. Use your RemoveWM account to login

#### 2. Download Videos

When browsing supported video websites:

1. Extension automatically detects videos on the page
2. Shows "Download" button next to video player
3. Click button to start download
4. Wait for processing to complete (progress shown)
5. Video automatically saves to download folder

### Advanced Features

#### Batch Download

Batch download videos on list pages:

1. Browse video list page (e.g., TikTok homepage)
2. Click extension icon to open panel
3. Check videos you want to download
4. Click "Batch Download" button

#### Custom Settings

Click extension icon ‚Üí Settings to configure:

- **Default Quality**: Choose 1080p, 720p or original

## Frequently Asked Questions

### Extension Cannot Detect Videos

**Possible Causes:**

1. Website updated page structure
2. Video is private or deleted
3. Extension needs updating

**Solutions:**

- Refresh page and retry
- Check for new version
- Report issue to us

### Slow Download Speed

**Optimization Tips:**

- Choose lower quality (e.g., 720p)
- Avoid downloading during peak hours
- Check network connection
- Consider upgrading membership (priority processing queue)

### Downloaded Videos Won't Play

**Check Items:**

1. Confirm download completed (check progress)
2. Check if file size is normal
3. Try opening with another player
4. Re-download video

### Extension Icon Not Showing

**Solution Steps:**

1. Check if extension is enabled
2. Restart Chrome browser
3. Reinstall extension

## Keyboard Shortcuts

Extension supports the following shortcuts:

| Shortcut | Function |
