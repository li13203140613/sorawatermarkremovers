# Chrome 扩展完整使用指南

RemoveWM Chrome 扩展让您可以直接在浏览器中一键下载无水印视频，无需复制链接到网站。

## 扩展特点

### 核心功能

- 🚀 **一键下载**：浏览视频时直接点击下载
- 🎯 **自动检测**：智能识别支持的视频平台
- ⚡ **快速处理**：后台自动去除水印
- 💾 **批量下载**：支持同时下载多个视频
- 🔒 **隐私保护**：本地处理，不上传数据

### 支持的网站

目前支持以下平台：

1. **抖音** - douyin.com
2. **小红书** - xiaohongshu.com
3. **快手** - kuaishou.com
4. **TikTok** - tiktok.com
5. **西瓜视频** - ixigua.com

## 安装步骤

### 方法一：从 Chrome 网上应用店安装（推荐）

1. 访问 [Chrome 网上应用店](#)
2. 搜索 "RemoveWM"
3. 点击"添加至 Chrome"
4. 在弹窗中点击"添加扩展程序"

### 方法二：手动安装开发者版本

如果您想使用最新的测试版本：

#### 步骤 1：下载扩展文件

```bash
# 下载扩展源码
git clone https://github.com/removewm/chrome-extension.git
cd chrome-extension
npm install
npm run build
```

#### 步骤 2：加载到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 开启右上角的"开发者模式"
4. 点击"加载已解压的扩展程序"
5. 选择 `build` 文件夹

## 使用教程

### 基础使用

#### 1. 登录账号

首次使用需要登录：

1. 点击浏览器工具栏的 RemoveWM 图标
2. 点击"登录"按钮
3. 使用您的 RemoveWM 账号登录

#### 2. 下载视频

当您浏览支持的视频网站时：

1. 扩展会自动检测页面上的视频
2. 在视频播放器旁边显示"下载"按钮
3. 点击按钮开始下载
4. 等待处理完成（会显示进度）
5. 视频会自动保存到下载文件夹

### 高级功能

#### 批量下载

在列表页面批量下载视频：

1. 浏览视频列表页面（如抖音首页）
2. 点击扩展图标打开面板
3. 勾选想要下载的视频
4. 点击"批量下载"按钮

#### 自定义设置

点击扩展图标 → 设置，可以配置：

- **默认画质**：选择 1080p、720p 或原画
- **保存路径**：自定义下载文件夹
- **文件命名**：设置文件名格式
- **自动下载**：检测到视频后自动开始下载

```javascript
// 设置示例
{
  "quality": "1080p",
  "savePath": "Downloads/RemoveWM",
  "fileFormat": "{platform}_{author}_{title}_{date}",
  "autoDownload": false
}
```

## 常见问题

### 扩展无法检测视频

**可能原因：**

1. 网站更新了页面结构
2. 视频是私密或已删除
3. 需要更新扩展版本

**解决方法：**

- 刷新页面重试
- 检查是否有新版本
- 向我们反馈问题

### 下载速度慢

**优化建议：**

- 选择较低的画质（如 720p）
- 避免高峰期下载
- 检查网络连接
- 考虑升级会员（优先处理队列）

### 下载的视频无法播放

**检查事项：**

1. 确认下载完成（查看进度）
2. 检查文件大小是否正常
3. 尝试用其他播放器打开
4. 重新下载视频

### 扩展图标不显示

**解决步骤：**

1. 检查扩展是否已启用
2. 重启 Chrome 浏览器
3. 重新安装扩展

## 快捷键

扩展支持以下快捷键：

| 快捷键 | 功能 |
|--------|------|
| `Alt + D` | 下载当前视频 |
| `Alt + S` | 打开扩展设置 |
| `Alt + H` | 查看下载历史 |
| `Ctrl + Shift + D` | 批量下载模式 |

可以在 `chrome://extensions/shortcuts` 自定义快捷键。

## 权限说明

扩展需要以下权限：

### 必需权限

- **storage**：保存设置和下载历史
- **downloads**：管理下载的文件
- **tabs**：检测当前标签页的视频

### 可选权限

- **notifications**：显示下载完成通知
- **contextMenus**：右键菜单快捷操作

### 隐私声明

我们承诺：

- ✅ 只在支持的网站运行
- ✅ 不收集浏览历史
- ✅ 不跟踪用户行为
- ✅ 仅上传必要的处理数据
- ✅ 下载记录只保存在本地
