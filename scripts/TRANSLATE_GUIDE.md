# 博客自动翻译功能使用指南

## 🌐 功能说明

现在创建博客时，如果您**不输入英文标题和描述**，工具会提示您进行翻译，您可以：
1. 让 AI 助手（如 Claude Code）帮您翻译
2. 手动输入翻译
3. 直接跳过（使用中文）

---

## 🚀 使用流程

### 示例演示

```bash
$ npm run blog

🚀 RemoveWM 博客文章创建工具（AI 增强版）

📝 文章标题（中文）: 如何提升视频编辑效率
📝 文章标题（英文，留空自动翻译）: [直接回车]

🤖 正在使用 AI 翻译标题...
💡 提示: 请在下方输入翻译结果，AI 助手会帮您翻译

==================================================
请翻译以下中文为英文:
标题: 如何提升视频编辑效率
==================================================

✍️  请输入翻译后的英文标题:
```

**这时您有几个选择：**

### 选择 1：使用 AI 助手翻译（推荐）⭐

1. **看到翻译提示后，不要输入任何内容**
2. **对 Claude Code 说**：
   ```
   请翻译这个标题为英文：如何提升视频编辑效率
   ```
3. **Claude Code 会回复**：
   ```
   How to Improve Video Editing Efficiency
   ```
4. **复制翻译结果，粘贴到命令行**
5. **继续**

### 选择 2：手动翻译

直接输入您自己的英文翻译：
```
✍️  请输入翻译后的英文标题: How to Improve Video Editing Efficiency
```

### 选择 3：跳过翻译

直接按回车跳过，工具会使用中文：
```
✍️  请输入翻译后的英文标题: [直接回车]
⚠️  警告: 未提供英文标题，将使用中文拼音作为 slug
```

## 💡 最佳实践

### 推荐工作流

1. **准备好中文内容**
   - 想好标题
   - 写好描述
   - 列出核心要点

2. **运行工具**
   ```bash
   npm run blog
   ```

3. **输入中文，英文留空**
   - 所有英文输入框直接回车

4. **等待翻译提示**
   - 看到翻译请求时
   - 让 AI 助手翻译
   - 复制粘贴结果

5. **完成创建**
   - 继续填写其他信息
   - 生成双语文章

### 效率提升技巧

#### 技巧 1：批量翻译

准备多个需要翻译的内容，一次性让 AI 翻译：

```
请帮我翻译以下内容为英文：

1. 标题：AI 视频生成技巧大全
2. 描述：全面介绍使用 AI 工具生成高质量视频的方法和技巧
3. 要点1：选择合适的 AI 视频工具
4. 要点2：编写有效的提示词
5. 要点3：视频质量优化
```

AI 会一次性给出所有翻译。

#### 技巧 2：保存常用翻译

创建一个翻译对照表：

| 中文 | 英文 |
|------|------|
| 视频处理 | Video Processing |
| AI 技术 | AI Technology |
| 使用教程 | Tutorial |
| 快速入门 | Quick Start |
| 最佳实践 | Best Practices |

#### 技巧 3：使用模板

准备好常用的英文模板：

```
标题模板：
- How to [动作] [对象]
- Complete Guide to [主题]
- [数字] Tips for [目标]

描述模板：
- Learn how to...
- Discover the best ways to...
- Complete guide on...
```

## ⚙️ 技术说明

### 工作原理

1. **检测英文是否为空**
   ```javascript
   if (!titleEn && titleZh) {
     // 提示用户需要翻译
   }
   ```

2. **显示翻译提示**
   - 清晰展示需要翻译的中文
   - 等待用户输入翻译结果

3. **接受翻译输入**
   - 用户可以输入 AI 给出的翻译
   - 也可以手动输入
   - 或者直接跳过

### 为什么不完全自动化？

**原因：**
1. **需要 API 密钥**：完全自动翻译需要调用翻译 API（如 OpenAI、Google Translate）
2. **成本考虑**：API 调用需要付费
3. **质量控制**：人工审核翻译结果更准确
4. **灵活性**：用户可以选择是否翻译

**当前方案优势：**
- ✅ 无需 API 密钥
- ✅ 无额外成本
- ✅ 质量可控
- ✅ 灵活方便

## 🎓 学习资源

### 翻译技巧

1. **技术术语保持一致**
   - AI → AI（不翻译）
   - 视频 → Video
   - 水印 → Watermark

2. **标题翻译原则**
   - 简洁明了
   - 包含关键词
   - 符合英文习惯

3. **描述翻译原则**
   - 完整传达意思
   - 使用主动语态
   - 避免中式英语

### 常用翻译模式

```
如何...  → How to...
...指南  → Guide to...
...技巧  → Tips for... / Techniques for...
最佳实践 → Best Practices
快速入门 → Quick Start
详细教程 → Complete Tutorial
```
